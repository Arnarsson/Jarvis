# Plan 08-01: Catch Me Up API

## Goal
Build the "catch me up on X" feature - contextual summaries from all data sources.

## Tasks

### 1. Context Gatherer Service
Create service that aggregates relevant data for a topic:
- Search screen captures for topic
- Search emails mentioning topic
- Search calendar events related to topic
- Search chat conversations about topic

### 2. Summary Generator
Use Claude API to synthesize gathered context:
- Chronological narrative of activity
- Key decisions and outcomes
- Open items and next steps
- People involved

### 3. API Endpoint
```
POST /api/catchup
{
  "topic": "jarvis project",
  "timeframe": "last_week",  // optional
  "depth": "detailed"  // summary | detailed
}
```

### 4. MCP Tool
Add `jarvis_catch_me_up` tool for Claude Code access.

## Files to Create/Modify
- `src/jarvis_server/catchup/gatherer.py` - Context aggregation
- `src/jarvis_server/catchup/summarizer.py` - LLM summarization
- `src/jarvis_server/api/catchup.py` - API endpoint
- `src/jarvis_server/mcp/tools.py` - Add MCP tool
